wslg electron linux qq

2022-12-28 2 min read

自行凑够wsl2前置环境,个人是在wt+ps7里面敲的,仅供参考
wsl --install
自动安装适用于Linux的Windows子系统并且默认安装Ubuntu的LTS版本
ubuntu
继续安装ubuntu,并且设置用户名和密码
我这边安装后是 Ubuntu 22.04.1 LTS,然后是更新一下软件
sudo apt update
记得输入密码验证
sudo apt upgrade
安装一些qq所需的依赖,当然未来它可能需要更多依赖,建议按照log补
sudo apt install libgtk-3-0 libnotify4 libnss3 libxss1 libxtst6 xdg-utils libatspi2.0-0 libsecret-1-0 libgbm1 libasound2
接着是安装QQ,Ubuntu可以选.deb安装,记得架构要选择符合自己的,这个是当时能下的x64架构的.deb包
wget https://dldir1.qq.com/qqfile/qq/QQNT/64bd2578/linuxqq_3.0.0-565_amd64.deb
然后用dpkg安装
sudo dpkg -i QQ-v2.0.3-543_x64.deb
不着急打开,配置一下编码和共享一下win的字体库给wsl,否则待会中文会乱码
sudo dpkg-reconfigure locales
使用空格键选中
en_US.UTF-8和zh_CN.UTF-8
使用Tab键选中Ok然后回车键确认
在下一个界面选择将这个设置为默认
en_US.UTF-8
编辑一下配置,micro可以换成vim等其他编辑器
sudo micro /etc/fonts/local.conf
复制粘贴以下内容进去

<?xml version="1.0"?>
<!DOCTYPE fontconfig SYSTEM "fonts.dtd">
<fontconfig>
    <dir>/mnt/c/Windows/Fonts</dir>
</fontconfig>

记得要保存编辑的内容,像micro你需要ctrl+s保存然后ctrl+q退出
现在你可以直接运行QQ了
qq
